runOnce :: (HasStats m, MonadLogger m, MonadIO m) => (a -> m ()) -> QueueRunner a -> m () Source #

Block until an item might be ready and then run (and remove) all ready items. This will sometimes run 0 items. It shouldn't ever run any items that are scheduled for the future, and it shouldn't forget any items that are ready.
